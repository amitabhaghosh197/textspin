Text Spinner jQuery Plugin

I want to SPIN with you.

We are adorable

Setting up the HTML

In order to get the text spinner to work we need to set up some markup to have it run smoothly

We need to set up span container that can be anything I have named mine 'ts-container' after that you need two other spans that spin/rotate the text.

You need to set them up as 'ts-word-one' and 'ts-word-two' 


			<p class="ts"> I want to 
			    <span id="ts-container">
				    <span class="ts-word-one">SPIN </span><span class="ts-word-two"></span> 
				</span> 
				with you.
			</p>

Setting up the javascript

We need to include the jquery library and the jquery-ui library in order to make the plugin work.

Setting up the CSS


			#ts-container-one {
				display:inline-block;
				position:relative;
				height:60px; //change this base on your font size
				width:165px; //change this base on your font size
			}
			/* these two are needed to make it work and display correctly */
			.ts-word-one {
				  display:block;
				  position:absolute;
			}
			.ts-word-two {
				display:none;
				position:absolute;
			}
		

		$(document).ready(function(){
			var words = ['SPIN','RUN', 'JUMP', 'PLAY']
			$('#ts-container').textSpin(words);
			var words = ['adorable', 'beautiful', 'glamorous', 'magnificent'];
			$("#ts-container-two").textSpin(words);
		});
